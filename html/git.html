Git

  • git init - расшифровывается как "Initialize", что переводится как инициализировать, с помощью нее создается новый локальный репозиторий в текущей директории.
  • git add - переводится как добавить, с помощью нее можно добавить изменения в файлах.
  • git commit - переводится как фиксировать.
  • git status - переводится как состояние, с помощью нее можно узнать текущее состояние рабочей директории.
  • git branch - переводится как ветка, с помощью нее можно увидеть список веток в директории и создать новую ветку
